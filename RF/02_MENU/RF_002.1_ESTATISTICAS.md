# RF_002.1 - Estatísticas Gerais

## VISÃO GERAL

Esta funcionalidade descreve a tela de dashboard de estatísticas do usuário, acessível através do dropdown de perfil no Menu Principal. Ela consolida dados de engajamento, performance e progresso de aprendizado para que o usuário monitore sua evolução no conhecimento geográfico.

## ÉPICO

Como usuário dedicado, quero visualizar estatísticas detalhadas do meu uso para entender quais regiões domino e quais preciso estudar mais.

## HISTÓRIA DE USUÁRIO

Como jogador, quero ver um resumo de quantas partidas joguei, qual modo de jogo eu mais utilizo e qual a minha porcentagem de conclusão do "conhecimento do mundo", permitindo uma visão clara dos meus pontos fortes e fracos.

## DETALHAMENTO DE CAMPOS E ELEMENTOS

Esta seção define as métricas e indicadores visuais da tela de estatísticas.

| Campo / Elemento | Tipo | Descrição / Comportamento esperado |
| :--- | :--- | :--- |
| **Cabeçalho de Perfil** | Texto / Foto | Exibe o Nick e a Foto do usuário (sincronizado com RF_001). |
| **Card: Engajamento Geral** | Container | Exibe o total de partidas jogadas e o tempo total acumulado no aplicativo. |
| **Indicador: Modo Favorito** | Texto / Ícone | Identifica qual dos modos (Bandeiras, Mapas, Quiz) possui o maior volume de horas ou partidas. |
| **Card: Conhecimento** | Barra de Progresso | Percentual de conclusão baseado no número de países/territórios acertados ao menos uma vez em relação ao total global. |
| **Recorde de Sequência** | Número | Exibe a maior sequência de acertos consecutivos (streak) já alcançada pelo usuário em uma única partida. |
| **Gráfico de Radar: Regiões** | Gráfico | Visualização comparativa de performance entre os continentes (Américas, África, Europa, Ásia, Oceania). |

## PRÉ-REQUISITOS

1. O sistema deve coletar e persistir os dados brutos de cada partida concluída na **RF_003.4**.

## DEPENDÊNCIAS

- RF_002 - Menu Principal do Aplicativo (ponto de origem).
- RF_003.4 - Tela de Resultados (fonte primária de dados de performance).

## FORA DE ESCOPO

- Ranking global de jogadores (Leaderboards) - será tratado em RF futura.
- Filtro de estatísticas por data (ex: "últimos 7 dias").

## COMPORTAMENTO ESPERADO

### Cenário 1: Visualização de Dados Iniciais

**Dado que** o usuário acabou de criar uma conta e ainda não jogou nada,
**Quando** acessar a tela de estatísticas,
**Então** o sistema deve exibir os campos com valor zero ou placeholders esteticamente agradáveis (ex: "Jogue sua primeira partida para ver dados aqui").

### Cenário 2: Atualização de Progresso Mundial

**Dado que** o usuário acertou 10 novas bandeiras em uma partida,
**Quando** retornar ao dashboard de estatísticas,
**Então** o card de progresso "Conhecimento" deve refletir o incremento proporcional à base total de países.

### Cenário 3: Navegação de Retorno

**Dado que** o usuário está visualizando suas estatísticas,
**Quando** clicar no botão "Voltar",
**Então** o sistema deve retornar o usuário para a tela do Menu Principal (RF_002).
